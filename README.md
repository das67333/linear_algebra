если rust не установлен: https://win.rustup.rs/

-> разархивировать и перейти в эту папку (linear_algebra)

Windows 10:
-> зажать shift, правой кнопкой мыши на пустом месте

Windows 11:
-> правой кнопкой -> Open in Windows Terminal

-> cargo run
